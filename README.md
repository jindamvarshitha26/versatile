# Versatile IT Solutions Website

This is a simple company website built using Flask, HTML, and CSS.

## Features
- Multiple pages (Home, About, Services, Contact)
- Responsive design
- Flask backend

## How to Run

1. Install requirements:
pip install -r requirements.txt

2. Run the app:
python app.py

3. Open in browser:
http://127.0.0.1:5000/

## Deployment
Deployed using Render.
